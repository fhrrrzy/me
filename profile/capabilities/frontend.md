# Frontend Stack

| Category | Technologies |
|----------|-------------|
| Frameworks | Vue 3 (Composition API), React 19, Next.js 16 (App Router) |
| Meta-frameworks | Nuxt, Inertia.js v2 |
| Styling | Tailwind CSS v4, CSS Modules |
| Component Libraries | shadcn-vue, shadcn/ui, Reka UI, base-ui, Filament |
| State | Pinia, React hooks, Inertia forms |
| Animation | GSAP, CSS transitions, scroll-driven animations |
| Charts | ECharts (dual-axis, real-time), ApexCharts |
| Maps | Leaflet |
| File Handling | FilePond, ExcelJS |
| Mobile/Cross-platform | Expo, Tamagui, Solito (React Native + Next.js) |
| Build | Vite, Webpack |
| PWA | Service workers, offline-first |
