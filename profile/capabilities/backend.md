# Backend Stack

| Category | Technologies |
|----------|-------------|
| Frameworks | Laravel 12, Filament v3, Next.js API routes |
| Auth | Laravel Fortify, NextAuth, Google OAuth (Socialite), JWT, 2FA |
| Real-time | Laravel Reverb (WebSockets), Server-Sent Events (SSE) |
| Queues | Redis-backed with priority levels (high/default/low) |
| AI/LLM | Multi-provider adapter (OpenAI, Anthropic, OpenAI-compatible) |
| Bots | Pyrogram (Telegram userbot), python-telegram-bot, WhatsApp JS |
