# Design Philosophy

- **Clarity over cleverness** — If a user has to think about how to use it, it's wrong
- **Speed is a feature** — Fast load, smooth transitions, instant feedback
- **Mobile-first always** — Most users are on phones, especially in Indonesia
- **Motion with purpose** — GSAP animations that guide attention, not distract
- **Component systems** — Reusable, consistent, maintainable
