## Naming Convention

- `hero-desktop.png` — Above-the-fold view at 1440×900
- `hero-mobile.png` — Above-the-fold view at 390×844
- `[feature]-[context].png` — Specific feature screenshots
- Use lowercase, kebab-case
- PNG preferred for UI screenshots
