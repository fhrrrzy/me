# PT Perkebunan Nusantara IV (Persero)

**Officer Data Support — Software Developer**

## What I Build

- CMMS dashboard connected to SAP ERP PM module — real-time equipment monitoring
- Excel validation webapp — dendrogram-based data unification for 71 factories
- Palmira Chat — internal AI workspace with multi-provider LLM support
- PicaRet — rubber plantation management system
- Data sync pipelines, backup automation, Docker deployments

## Why It Matters

Turning scattered, manual processes into unified digital systems. Plant managers get real-time visibility. Data entry teams get automated validation. The entire operation moves from paper/Excel chaos to structured, queryable data.
