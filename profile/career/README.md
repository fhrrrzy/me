## The Arc

Started as a freelancer building websites and Telegram bots at 18. Went through a formal CS degree while continuing to ship real projects. Moved from startup (Cowalaid) to government (Diskominfo) to enterprise (PTPN IV), gaining depth at each stage. Now building systems that serve thousands of users across 71 factories while co-running a SaaS product on the side.

Each role added a layer:
- Freelance → independence, client management, shipping fast
- Cowalaid → team collaboration, design systems, API patterns
- Diskominfo → government scale, full lifecycle ownership
- PTPN IV → enterprise complexity, SAP integration, data at scale
